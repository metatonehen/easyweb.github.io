Introducción

Esta guía proporciona todos los detalles necesarios para implementar el diseño de METATONEHEN en WordPress, manteniendo la coherencia visual y la organización basada en el Cubo de Metatrón. A continuación encontrarás información detallada sobre la paleta de colores, la tipografía, los componentes principales y el código CSS necesario.

Nota importante: Para implementar correctamente este diseño en WordPress, se recomienda utilizar un tema personalizable como Astra, Divi, Elementor o similar, que permita personalizar completamente los estilos CSS y la estructura del sitio.

Implementación en WordPress

Para implementar esta paleta de colores en WordPress:

  1. Ve a Apariencia > Personalizar en tu panel de WordPress
  2. Busca la opción de colores o tema (varía según el tema que estés utilizando)
  3. Define los colores principales y secundarios según la paleta mostrada arriba
  4. Si tu tema no permite personalizar todos estos colores, utiliza el siguiente código CSS personalizado
:root { --dark: #0a0722; --indigo: #4A1A9E; --indigo-dark: #2A0A5D; --violet-light: #a675c8; --gold: #D4AF37; --gold-light: #F5D76A; --off-white: #F5F5F5; --white: #FFFFFF; } body { background-color: var(--dark); color: var(--off-white); } h1, h2, h3, h4, h5, h6 { color: var(--gold); } a { color: var(--gold); } a:hover { color: var(--gold-light); }

Implementación en WordPress

Para implementar la tipografía en WordPress:

  1. Instala el plugin "Google Fonts Typography" o similar
  2. Configura Montserrat como fuente para títulos (h1-h6)
  3. Configura Lato como fuente para el cuerpo de texto
  4. Alternativamente, agrega el siguiente código al CSS personalizado del tema
/* Importar fuentes de Google */ @import url('https://fonts.googleapis.com/css2?family=Lato:ital,wght@0,400;0,700;1,400&family=Montserrat:wght@400;500;700&display=swap'); /* Configuración de tipografía */ :root { --heading-font: 'Montserrat', sans-serif; --body-font: 'Lato', sans-serif; } body { font-family: var(--body-font); line-height: 1.6; } h1, h2, h3, h4, h5, h6 { font-family: var(--heading-font); font-weight: 700; } .greek-text { font-family: var(--body-font); font-style: italic; color: var(--violet-light); } /* Estilos específicos para títulos */ h1 { font-size: 2.5rem; background: linear-gradient(to right, var(--gold), var(--gold-light)); -webkit-background-clip: text; -webkit-text-fill-color: transparent; text-transform: uppercase; letter-spacing: 2px; } h2 { font-size: 2rem; margin-bottom: 20px; } h3 { font-size: 1.5rem; color: var(--gold); }

Cubo de Metatrón y Organización Hexagonal

El Cubo de Metatrón es un elemento central en el diseño de METATONEHEN, no solo como elemento visual, sino como estructura organizativa para los cursos y secciones del sitio.

Los vértices del Cubo de Metatrón representan las principales categorías de cursos y están posicionados en una distribución hexagonal alrededor del centro. Esta disposición se mantiene coherente en todo el sitio.

Implementación en WordPress

Para implementar el Cubo de Metatrón y su organización en WordPress:

  1. Usa un constructor de páginas como Elementor, Divi o Visual Composer
  2. Crea una sección con posición relativa para contener el cubo
  3. Añade la imagen SVG del Cubo de Metatrón como fondo
  4. Crea elementos HTML personalizados para cada vértice
  5. Aplica las clases de posicionamiento para cada vértice
  6. Vincula cada vértice a la categoría o página correspondiente
Consejo: Si tu tema WordPress no permite un control tan preciso de la posición, considera crear un HTML personalizado usando un plugin como "Custom HTML Widget" o "Shortcode HTML" para insertar este componente.

Alternativa: Utiliza la imagen del Cubo de Metatrón como mapa de imagen y define áreas clickeables para cada vértice usando el elemento <map> de HTML.

Guía de Exportación a WordPress

Sigue estos pasos para implementar completamente el diseño en WordPress:

  1. Preparación

    Instala WordPress y elige un tema altamente personalizable como Astra, Divi, o usa Elementor Pro para mayor flexibilidad.

  2. Importación de Archivos Multimedia

    Sube todas las imágenes y archivos SVG a la biblioteca multimedia de WordPress:

    • Instala el plugin "Safe SVG" para permitir la carga de archivos SVG
    • Ve a Medios > Añadir nuevo y sube los archivos SVG y de imagen

  3. Configuración de Tipografía y Colores

    Define la paleta de colores y tipografía en la personalización del tema o mediante CSS personalizado.

  4. Creación de Elementos del Cubo de Metatrón

    Utiliza un constructor de páginas (como Elementor) o un bloque HTML personalizado para crear la estructura del Cubo de Metatrón con sus vértices interactivos.

  5. Implementación de Animaciones

    Añade el código JavaScript y CSS para las animaciones en la sección de código personalizado de tu tema.

  6. Creación de Páginas y Categorías

    Crea las páginas principales (inicio, cursos, sesiones, membresías) y configura las categorías para los cursos que coincidan con los vértices del Cubo de Metatrón.

  7. Pruebas Responsivas

    Prueba el sitio en diferentes dispositivos para asegurarte de que el diseño se adapta correctamente, especialmente la estructura del Cubo de Metatrón en dispositivos móviles.

Plugins Recomendados para WordPress

  • Elementor Pro: Para crear diseños personalizados sin código
  • Safe SVG: Para permitir archivos SVG en WordPress
  • WP Rocket: Para optimizar el rendimiento del sitio
  • Yoast SEO: Para optimización de motores de búsqueda
  • WooCommerce: Si necesitas vender cursos o membresías
  • LearnDash o LifterLMS: Para la gestión avanzada de cursos
